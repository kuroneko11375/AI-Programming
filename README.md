## 開發方式

1. **確認需求或想法**  
   - 把要做的事情先整理清楚，再交給 AI 協助產生初版程式碼。  

2. **程式產生與調整**  
   - 依靠 AI 生成範例，然後再根據情況嘗試修改或修正。  

3. **測試與修正**  
   - 基本測試能否執行，遇到問題時再請 AI 幫忙分析。  

4. **版本紀錄**  
   - 將可以正常運作的版本上傳到 GitHub，作為紀錄與整理。

- 提供除錯方向與解釋
